Digital Sign In/Out System

2022-2024, Software Engineering Project

Flutter Firebase Mobile Development

As part of a 3-person team at Mass Academy, I contributed to the creation of a digital sign-in/out system designed to replace an outdated paper-based process. The project involved building a system architecture that seamlessly integrated local hardware (Raspberry Pi), MySQL data storage, and a NodeJS website.

My role focused on designing the MySQL database schema, which was structured to handle over 1000 unique student schedules. These schedules vary each school day and are automatically synchronized with the calendar to ensure accurate tracking of student attendance.

I also programmed the Raspberry Pi to decode a proprietary HID RFID card bit format, enabling it to interact with RFID readers and student ID cards, making the sign-in/out process both efficient and secure.

Additionally, I took responsibility for filing HECVAT forms and collaborating with the WPI Information Security team to ensure that our system met FERPA privacy and security standards, protecting student data at all times.
