Balancing Inverse Pendulum

Spring 2026, Programming an inverse pendulum robot to self balance using differential equations.

MATLAB Simulink Arduino Controls

About

The goal of this project was to program an inverse pendulum robot to self-balance using a combination of control theory and ordinary differential equations.

Documentation

The detailed mathematical work and processes are provided here in our project documentation. We cover an overview of the project, modeling the system with 3 and 5 poles, calculating the poles, and simulating the response with Simulink.
